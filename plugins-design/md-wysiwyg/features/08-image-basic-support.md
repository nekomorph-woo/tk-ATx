# 图片基础支持

## 功能描述

支持插入本地图片、粘贴图片、拖拽图片到编辑器，并生成标准 Markdown image 语法。

## 为什么做？

图片是 Markdown 文档常见内容。即使暂不做上传服务，本地路径和相对路径支持也能覆盖大量个人笔记和项目文档场景。

## 收益

- 提升图文文档写作体验。
- 为后续图片上传服务集成留下接口。
- 增强 README、教程、设计文档编写能力。

## 基础设施支持情况

- 现有依赖：Milkdown upload plugin、clipboard plugin。
- 可能新增代码：本地文件复制/引用策略、相对路径计算、drop/paste handler、图片 NodeView 样式。
- 可能新增依赖：如需要生成唯一文件名，可直接用 Node API；不必新增依赖。

## 实现要点

- 先支持插入已有本地图片路径。
- 粘贴/拖拽图片时复制到文档旁的 `assets/` 或可配置目录。
- Markdown 使用相对路径。
- 图片点击后可编辑 alt、src、title。

## 风险与注意事项

- 文件复制策略要避免覆盖同名文件。
- 远程图片、本地绝对路径、相对路径的行为需要明确。
