# 表格编辑增强

## 功能描述

为 GFM 表格提供可视化编辑能力，包括插入表格、添加/删除行列、调整对齐方式、在单元格间键盘导航。

## 为什么做？

表格是 Markdown 中最难手写、最适合 WYSIWYG 的结构之一。当前即使能解析表格，如果缺少图形化编辑，体验仍接近源码编辑。

## 收益

- 大幅提升 GFM 表格可用性。
- 让插件更接近 Typora 类编辑体验。
- 对技术文档、参数对比、清单类内容价值很高。

## 基础设施支持情况

- 现有依赖：Milkdown GFM preset、ProseMirror table 相关能力。
- 可能新增代码：表格操作 command、单元格选区状态、行列操作浮层、对齐菜单。
- 可能新增依赖：优先使用 Milkdown/GFM 已带能力；如不足，再评估 `prosemirror-tables` 直接能力。

## 实现要点

- Slash Command 和工具栏提供“插入表格”入口。
- 表格选中时显示行列操作按钮。
- 支持 Tab / Shift+Tab 在单元格间移动。
- 序列化必须保持标准 GFM 表格。

## 风险与注意事项

- 表格结构复杂，容易影响撤销/重做和选区。
- 大表格性能和样式溢出需要单独验证。
