# Slash Command

## 功能描述

在编辑器中输入 `/` 后弹出插入菜单，支持快速插入标题、列表、引用、代码块、表格、数学公式、Mermaid 图表、任务列表等 Markdown 结构。

## 为什么做？

WYSIWYG 模式隐藏了大量 Markdown 语法，用户需要一个低记忆成本的入口来发现和插入结构化内容。Slash Command 是现代文档编辑器中最自然的块级操作入口。

## 收益

- 显著提升新用户可发现性。
- 减少记忆 Markdown 语法和菜单路径的成本。
- 为后续表格、公式、图表等复杂块提供统一入口。

## 基础设施支持情况

- 现有依赖：Milkdown command、input rule、ProseMirror plugin。
- 可能复用：`@milkdown/kit/plugin/slash` 或自定义 tooltip/plugin。
- 可能新增代码：菜单 UI、命令注册、键盘导航、过滤搜索、插入命令适配层。
- 可能新增依赖：如需要更完整浮层定位，可考虑 `@floating-ui/dom`，但 Mermaid 依赖链里已有相关包，优先评估是否可复用。

## 实现要点

- 输入 `/` 时在当前光标位置显示菜单。
- 支持上下键、回车、Esc、鼠标点击。
- 插入后删除 `/` 查询文本并创建对应 Markdown 节点。
- 菜单项按常用程度排序，并支持关键词过滤。

## 风险与注意事项

- 需要避免和普通文本中的 `/` 冲突。
- 中文输入法组合态下不要误触发。
- 菜单定位要适配滚动容器和不同主题。
