# 数学公式编辑增强

## 功能描述

增强 KaTeX 公式编辑体验，支持源码/预览切换、错误提示、块级公式工具按钮和常用符号插入。

## 为什么做？

数学公式是 Markdown 编辑器中很容易暴露体验差异的功能。用户需要既能看到渲染结果，又能快速修改 LaTeX 源码。

## 收益

- 提升学术笔记、技术推导、算法文档写作体验。
- 减少公式错误时的排查成本。
- 强化插件相对普通 Markdown preview 的优势。

## 基础设施支持情况

- 现有依赖：KaTeX、remark-math、自定义 math node schema 和 NodeView。
- 可能新增代码：公式浮层、错误详情、常用符号菜单、block/inline 插入 command。
- 暂不需要新增第三方依赖。

## 实现要点

- 行内公式点击进入源码编辑。
- 块级公式提供固定源码/预览切换。
- KaTeX 报错时显示可读错误，不吞掉原公式。
- 工具栏和 Slash Command 能插入行内/块级公式。

## 风险与注意事项

- 输入框和 ProseMirror 选区同步要稳定。
- 公式内特殊字符需要保证序列化不被错误转义。
